columns

columns is shorthand for column-count and column-width.

1) Fixed column count

Demonstrates columns: 3; for a fixed number of columns.

CSS Multi-column layout lets content flow naturally from one column to the next. It is handy for readable long-form text in limited space. The browser balances content across columns while preserving normal document flow. You can also control spacing with column-gap and draw a divider with column-rule.

2) Width-based responsive columns

Demonstrates columns: 14rem auto;: target width per column, browser decides how many fit.
